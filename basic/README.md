# basic

定位：Java 入门基础语法与常用操作

子模块概览：
- variable：变量、常量、基本类型与包装类
- operator：算术/赋值/关系/逻辑/位运算与优先级
- control-flow：分支与循环（if/else、switch、for/while/do-while、break/continue）
- array：一维/多维数组与 Arrays 工具类
- string：String / StringBuilder / StringBuffer 的创建、拼接与性能

运行：
- 构建当前父模块及子模块：`mvn -pl basic -am test`
- 在子模块内执行：`mvn test`
